Slide 3 Explaining the code:

This code takes the variable chosen by the user from the mtcars dataset and stores it in x. U then generates a histogram bins using the chosen slider valus and draws the histogram with labels and a title. Finally, it calculates and shows the summary statitics.

# In server: computes stats for the selected variables
  x <- mtcars[[input$var]]
        bins <- seq(min(x, na.rm = TRUE),
                    max(x, na.rm = TRUE),
                    length.out = input$bins + 1)

        # draw the histogram with the specified number of bins
        hist(x, breaks = bins, col = 'blue', border = 'white',
             xlab = input$var,
             main = input$var)
        # outputs the mean, sd, and other variables
                         paste("Variable:", input$var,
                                "| n =", length(x),
                                "| mean =", round(mean(x), 2),
                                "| sd =", round(sd(x), 2))
